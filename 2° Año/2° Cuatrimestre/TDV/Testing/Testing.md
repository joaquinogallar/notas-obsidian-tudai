# Qué es?
Es la tarea de probar que un programa realiza lo que se supone que debe hacer. Aun haciendo lo esperado, puede contener fallas.

Test **NO** exitoso -> No detecta errores
Test exitoso -> Detecta errores.

# Tipos de Test:
- **[[Test Unitario]]**.
- Test de Integración.
- Test Funcional.
- Test de Sistema.
- Test de Aceptación.
