# ¿Qué es?
- Es un **Platform Independent Language**.
- **POO**:
	- Encapsulamiento/Encapsulation.
	- Abstracción/Abstraction.
	- Herencia/Inheritance.
	- Polimorfismo/Polymorphism.
- Portabilidad (**WORA**)
	- **W**rite **O**nce **R**un **A**nywhere.
